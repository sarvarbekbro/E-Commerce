# Boom Shop 🛒

Boom Shop is a simple e-commerce web application built with Node.js, Express, MongoDB, and Handlebars.

## Features

- User authentication with JWT
- Secure password hashing using bcrypt
- Add and view products
- Protected routes
- Flash messages for error handling

## Tech Stack

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Handlebars
- Bootstrap

## Installation

Clone the repository:

git clone https://github.com/yourusername/boom-shop.git](https://github.com/sarvarbekbro/E-Commerce.git

npm install

npm run dev
